Installation

Retour à l'index

Cette rubrique indique les étapes pour l'exploitation des sources. Elle est réservée aux développeurs.

Prérequis

  • Java 1.6
  • Maven 2

Initialisation

Installation :
>mvn clean install

(Optionnel pour créer l'environnement de l'IDE intellij avec le framework codjo)
>idea

(Optionnel pour créer l'environnement de l'IDE intellij sans le framework codjo)
>mvn idea:idea

Configuration pour les tests-releases

Alimenter le settings.xml avec le profil suivant : Profil

Exécution des tests

Si le framework codjo est disponible, les tests releases (tests ihm) peuvent être lancés :
>mvn clean install -Dprocess=integration

Sinon :_(
>mvn test

Générations des rapports de qualimétrie de code

>cd ${artifactId}
>mvn site
>mvn dashboard:dashboard

Le fichier de rapport se situe ensuite dans le repertoire target/site/dashboard-report.html
